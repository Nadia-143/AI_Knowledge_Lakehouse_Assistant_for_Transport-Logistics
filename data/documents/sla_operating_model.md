# SLA Operating Model

Each agency must submit datasets based on the agreed frequency.
Monthly datasets must be submitted before the due day defined in the SLA template.
If a dataset is late, the platform marks it as SLA breach.
If quality score is below the minimum threshold, the dataset is rejected by the quality gate.

Escalation:
1. Notify Data Steward.
2. Notify Data Governance Lead.
3. Request corrective action plan.
